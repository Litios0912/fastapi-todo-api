# Contributing

Gracias por tu interes en contribuir.

## Reportar bugs

Abre un Issue describiendo el problema y como reproducirlo.

## Enviar PRs

1. Fork el repositorio
2. Crea una rama: `git checkout -b mi-cambio`
3. Ejecuta los tests: `pytest`
4. Commit con mensaje claro
5. Push y abre un Pull Request

## Requisitos

- Los tests deben pasar
- El codigo debe seguir el estilo existente
- Agrega tests si introduces nueva funcionalidad
